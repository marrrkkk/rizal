# Chapter Background Music

This directory contains background music files for each chapter of the Rizal learning application.

## Music Files Needed

Place the following MP3 files in this directory:

1. **chapter1-childhood.mp3** - Gentle, playful music for childhood chapter
2. **chapter2-education.mp3** - Inspiring, academic music for education chapter
3. **chapter3-adventure.mp3** - Adventurous, travel music for studies abroad
4. **chapter4-literary.mp3** - Dramatic, literary music for Noli Me Tangere
5. **chapter5-heroic.mp3** - Heroic, patriotic music for return to Philippines
6. **chapter6-legacy.mp3** - Reflective, inspirational music for legacy chapter

## Recommended Sources for Royalty-Free Music

- **YouTube Audio Library**: https://www.youtube.com/audiolibrary
- **Free Music Archive**: https://freemusicarchive.org/
- **Incompetech**: https://incompetech.com/music/royalty-free/
- **Bensound**: https://www.bensound.com/
- **Purple Planet**: https://www.purple-planet.com/

## Music Guidelines

- **Duration**: 2-5 minutes (will loop automatically)
- **Format**: MP3, 128-192 kbps
- **Volume**: Normalized to prevent sudden loud sounds
- **Mood**: Should match the chapter theme
- **Style**: Instrumental, non-distracting background music

## Temporary Solution

If you don't have music files yet, the application will work without them. The music player will simply log a warning to the console.

To disable music completely, you can comment out the `<MusicControl>` component in each chapter file.
